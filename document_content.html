数值和公式可能会随着版本变化

悟性

在人物设定结束之后，悟性不能改变。悟性的高低会影响武功规划。

仁义

范围是-100到100之间，大部分情况下，满足以下条件之一，对方就能接受你：

  1. 你的仁义大于等于50时，对方仁义为正。

  2. 你的仁义小于等于-50，对方仁义为负。

  3. 双方仁义相差不超过50。

有一些特殊情况，比如：

  1. 黄蓉就算仁义刷到-100，仁义100的郭靖也愿意接受她。

  2. 欧阳锋就算仁义刷到100，仁义100的郭靖也不愿意接受他。

武功威力

武功威力与最终造成的伤害成正比。若只考虑武功威力，一次攻击造成的伤害为

武功威力 / 20

对局时间

9000帧后，对局会被判定为平局。为了尽量减少平局，各武功伤害会随着时间增加，比例为：

当前帧数 / 3000

即对局时间过去三分之一时，各角色伤害已翻倍。对局时间过去三分之二时，各角色伤害已是初始值3倍。

内力

内力不足时，伤害只有正常值3成。所以有时候内力不足时会自动选择威力弱、消耗内力低的武功。

攻击和防御

攻击和防御对最终伤害的影响为：

(攻方攻击 + 30) / (守方防御 + 30)

兵器值（即拳掌、御剑、耍刀、奇门）

兵器值对伤害的影响为：

(攻方兵器值 + 100) / (守方兵器值 + 100)

速度

速度值会影响移动速度，每帧的移动距离为：

14 - 9 * exp(-0.005 * 速度)

速度值会影响攻击速度，不考虑其他影响（如中毒），每10000帧的攻击次数为：

333 - 300 * exp(-0.01 * 速度)

附加内伤

攻方的“附加内伤”会增加守方对应的内伤值，内伤值范围是0-1000。内伤值会增加所受到的伤害，比例为：

内伤值 / 1000

内伤回复

角色的“内伤回复”会不断减少该角色的内伤值，每3帧减少

内伤回复

附加中毒

攻方的“附加中毒”会增加守方对应的中毒值，中毒值范围是0-1000。中毒值会降低攻击速度，比例为：

中毒值 / (中毒值 + 1000)

中毒回复

角色的“中毒回复”会不断减少该角色的中毒值，每3帧减少

中毒回复

暴起一击

攻方的“暴起一击”有概率造成2倍暴击伤害，概率为：

暴起一击 / (100 + 暴起一击)

左右开弓

有一定概率攻击两次（即互搏），概率为：

左右开弓 / 100

借力打力

攻方的“借力打力”会把上次攻击之后受到的伤害按比例返还给其中一个守方，比例为：

借力打力 / (100 + 借力打力)

以虚击实

攻方的“以虚击实”会在伤害计算时，无视守方部分防御，比例为：

以虚击实 / (70 + 以虚击实)

以巧胜拙

攻方的“以巧胜拙”会在伤害计算时，无视守方部分兵器值，数值为：

100 - 5000 / (50 + 以巧胜拙)

劲雄凝重

攻方的“劲雄凝重”会在伤害计算时，增加最终伤害，比例为：

劲雄凝重 / (100 + 劲雄凝重)

出其不意

攻方的“出其不意”有一定概率会让守方面对这次攻击时，无法触发“料敌机先”、“闪躲趋避”、“借招还招”、“真气护体”、“内力反震”、“倒刺护体”，概率为：

出其不意 / (100 + 出其不意)

内功卸力

守方的“内功卸力”会降低受到的伤害，比例为：

内功卸力 / (100 + 内功卸力)

宝甲护体

守方的“宝甲护体”会降低受到的伤害，比例为：

宝甲护体 / (100 + 宝甲护体)

闪躲趋避

守方的“闪躲趋避”有一定概率闪避某次攻击，概率为：

闪躲趋避 / (100 + 闪躲趋避)

借招还招

守方的“借招还招”在受到攻击后，有一定概率以相同武功反击攻方，概率为：

借招还招 / (100 + 借招还招)

反击时，武功等级按照被反击方算，反击方对应招式的兵器值额外增加：

借招还招 * 2

其他按照反击方攻击被反击方计算（比如，使用反击方的攻击和被反击方的防御），被反击方仍然可触发“闪躲趋避”等效果。

牵引挪移

守方的“牵引挪移”在受到攻击时，有一定概率将攻击目标转移到附近的敌人身上，概率为：

牵引挪移 / (100 + 牵引挪移)

转移后，新目标仍然可触发“闪躲趋避”等效果。伤害计算时，守方数值用新目标的数值。

内力反震

守方的“内力反震”在受到攻击后，有一定概率让攻击方也受到相同的伤害，概率为：

内力反震 / (100 + 内力反震)

倒刺护体

守方的“倒刺护体”在受到攻击距离小于等于400的拳掌攻击后，有一定概率让攻击方也受到相同的伤害，概率为：

倒刺护体 / (100 + 倒刺护体)

料敌机先

守方的“料敌机先”在受到攻击时，有一定概率触发特殊效果无视该次攻击，概率为：

料敌机先 / (100 + 料敌机先)

真气护体

守方的“真气护体”在受到攻击时，有一定概率扣除内力而不是生命值，概率为：

真气护体 / (100 + 真气护体)

内力不足时，会把内力扣到0，剩下伤害扣除生命值。

积蓄劲力

攻方的“积蓄劲力”会随着时间蓄力，蓄满后触发“蓄力一击”，造成1.5倍伤害。“积蓄劲力”对蓄力速度的影响为：

积蓄劲力

内力吸取

在攻击和被攻击时吸取内力，吸取的内力值为：

内力吸取

起死回生

在濒死时概率触发，回满全部生命。概率为：

起死回生 / 100

可重复触发。

真气反噬

在攻击时概率触发，取消该次攻击，并扣除10%当前内力。触发概率为：

真气反噬 / (100 + 真气反噬)

“真气反噬”小于等于0时，不会触发反噬。
